 Modify pom.xml for project name and maybe group.

 Make Configurations:
 1. Maven Configuration:
   - Click on Maven slider on the left of the screen
   - When Maven window opens, click on Hammer icon
   - In new opened window expand Maven (on the left) and click on Runner
   - Select checkBox for "Delegate IDE build/run actions to Maven"

2. Execution configuration:
  - first build project by pressing Hammer icon
  - then click on Add Configuration,
  - Select Jar application
  - In Jar Path dropdown go to ./target and select the .jar file in it
